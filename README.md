# firebase-esp8266-dht11

Code for this tutorial : https://www.youtube.com/watch?v=wlUjsAODKqE&t=1911s  
Wiring:  
NodeMCU D4 - DHT11 Dat  
NodeMCU 5V - DHT11 Vcc   
NodeMCU Gnd - DHT11 Gnd  
NodeMCU D1 - Led 1  
NodeMCU D2 - Led 2
